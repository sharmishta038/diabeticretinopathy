## ğŸ“Œ Project Overview

Diabetic Retinopathy (DR) is a severe diabetes complication that can lead to permanent vision loss if not detected early.

This project presents an **Automated Retinal Image Analysis System (ARIAS)** that uses **Deep Learning (CNN - Xception Architecture)** to classify retinal fundus images into five severity levels:

- No DR
- Mild DR
- Moderate DR
- Severe DR
- Proliferative DR

The system integrates a trained deep learning model with a **Flask Web Application** and **IBM Cloudant Database** for secure user authentication and real-time predictions.
